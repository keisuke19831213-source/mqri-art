# 量子曼荼羅 — Quantum Mandala(展示版)

`ccf5-gallery/art05` をベースにした **iPad / スマホ展示用フォーク**。
時間依存シュレーディンガー方程式をGPUで実時間解法し、神聖幾何学ポテンシャルの中で波束が干渉・回折する様子を描く(WebGL2・単一HTML・外部依存なし)。

## 元作品との違い(タッチ展示対応)

- **タッチツールバー** — 画面下部に大きなボタン群(44px以上のタップ領域)
  - 「波を放つ / 障壁を描く」モード切替(右ドラッグの代替)
  - 観測(波束の収縮)・リセット・障壁消去・配色・速度(キーボードの代替)
  - 幾何プリセット8種は横スクロールのチップで切替
- **マルチタッチ** — 最大6本の指で同時に波束を放てる(照準ガイドも指ごとに表示)
- **スリープ防止** — Screen Wake Lock API(iPadOS 16.4+)。タブ復帰時に自動再取得
- **誤操作防止** — ピンチズーム・ダブルタップズーム・ラバーバンドスクロール・長押しコールアウトをすべて抑止
- **アトラクトモード** — 90秒間操作がないと、既存の自動波束に加えて幾何プリセットを40秒ごとに自動巡回
- **レイアウト** — 描画プレートがツールバーを避けて配置。セーフエリア(ノッチ・ホームバー)対応
- **全画面** — ツールバーの「全画面」ボタンでブラウザのまま全画面化(Fullscreen API)。ホーム画面Webアプリとして起動した場合や非対応端末(iPhoneのSafari)ではボタンは自動的に隠れる

## 展示セットアップ(iPad)

**完全な全画面(時計・電池・終了ボタンも出ない)にするには「ホーム画面Webアプリ+アクセスガイド」で運用する。**
Safariの「全画面」ボタン(Fullscreen API)は手軽だが、iPadOSの仕様でステータスバー(時計・電池)と「✕」終了ピルはWeb側から消せず、✕は画面に触れるたびに再表示されるため、常設展示には不向き。

**手順(推奨・完全全画面)**
1. Safariで https://art.mqri.or.jp/quantum-mandala/ を開く
2. 共有メニュー →「ホーム画面に追加」
3. アクセスガイドを有効化(初回のみ):
   設定 → アクセシビリティ → **アクセスガイド** をオン → パスコードを設定 → 「画面の自動ロック」を「なし」に
4. 設定 → 画面表示と明るさ → **自動ロックを「なし」**
5. ホーム画面に追加したアイコンから作品を起動(Safari UIなしで立ち上がる)
6. トップボタン(またはホームボタン)を**3回押し** → 「開始」→ **ステータスバーも消えて完全全画面のキオスクになる**
   - 終了するときも3回押し → パスコード入力
7. 電源に接続したまま展示する

**簡易な代替(短時間のデモ向け)**
ツールバーの「全画面」ボタン(iPadOS 16.4以降のSafari / Chrome)。時計・電池と✕は残る。

## ローカル確認

```sh
python3 -m http.server 8642
# → http://localhost:8642
```

単一HTMLなので `index.html` を直接開いても動作する(WebGL2 + EXT_color_buffer_float 必須)。

## 操作(デスクトップは従来どおり)

ドラッグ=波束 / 右ドラッグ・Shift+ドラッグ=障壁 / M=観測 / 1–8=幾何 / SPACE=停止 / R=リセット / X=障壁消去 / C=配色 / B=輝き / ホイール=時間速度
