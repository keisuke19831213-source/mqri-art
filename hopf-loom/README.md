# HOPF LOOM — S³の織物(展示版)

`ccf5-gallery/art04` をベースにした **iPad / スマホ展示用フォーク**。
四次元球面 S³ の Hopf ファイブレーションをステレオ投影で三次元へ降ろし、四次元正多胞体(正五胞体〜正六百胞体)を重ねて実時間描画する(WebGL2・単一HTML・外部依存なし)。

## 元作品との違い(タッチ展示対応)

- **マルチタッチ・ジェスチャ**(マウス＋キーボード操作の置き換え)
  - **1本指ドラッグ** — 視点回転(3D)
  - **2本指ピンチ** — ズーム(ホイールの代替)
  - **2本指ドラッグ** — 四次元回転(Shift+ドラッグの代替)
- **タッチツールバー** — 画面下部に大きなボタン群(44px以上のタップ領域)
  - 多胞体4種(正五胞体 / 正八胞体 / 正二十四胞体 / 正六百胞体)を横スクロールのチップで切替
  - レイヤー表示切替(ファイバー / 多胞体 / 塵)・静止/再生・視点リセット
- **スリープ防止** — Screen Wake Lock API(iPadOS 16.4+)。タブ復帰時に自動再取得
- **誤操作防止** — ピンチズーム・ダブルタップズーム・ラバーバンドスクロール・長押しコールアウトをすべて抑止
- **アトラクトモード** — 90秒間操作がないと多胞体を45秒ごとに自動巡回(視点はもともと自動回転)
- **レイアウト** — タイトル/ステータスを上部帯に、操作を下部ツールバーに分離。セーフエリア(ノッチ・ホームバー)対応
- **全画面** — ツールバーの「全画面」ボタンでブラウザのまま全画面化(Fullscreen API)。ホーム画面Webアプリ起動時や非対応端末(iPhoneのSafari)ではボタンは自動的に隠れる

## 展示セットアップ(iPad)

**完全な全画面(時計・電池・終了ボタンも出ない)にするには「ホーム画面Webアプリ+アクセスガイド」で運用する。**
Safariの「全画面」ボタン(Fullscreen API)は手軽だが、iPadOSの仕様でステータスバー(時計・電池)と「✕」終了ピルはWeb側から消せないため、常設展示には不向き。

**手順(推奨・完全全画面)**
1. Safariで https://art.mqri.or.jp/hopf-loom/ を開く
2. 共有メニュー →「ホーム画面に追加」
3. アクセスガイドを有効化(初回のみ): 設定 → アクセシビリティ → **アクセスガイド** をオン → パスコードを設定
4. 設定 → 画面表示と明るさ → **自動ロックを「なし」**
5. ホーム画面に追加したアイコンから作品を起動(Safari UIなしで立ち上がる)
6. トップボタン(またはホームボタン)を**3回押し** → 「開始」→ **ステータスバーも消えて完全全画面のキオスクになる**
   - 終了するときも3回押し → パスコード入力
7. 電源に接続したまま展示する

**簡易な代替(短時間のデモ向け)**
ツールバーの「全画面」ボタン(iPadOS 16.4以降のSafari / Chrome)。時計・電池と✕は残る。

## ローカル確認

```sh
npx serve .
# → http://localhost:3000
```

単一HTMLなので `index.html` を直接開いても動作する(WebGL2 推奨。EXT_color_buffer_float が無い環境では自動でLDRにフォールバック)。

## 操作(デスクトップは従来どおり)

ドラッグ=視点 / Shift+ドラッグ=4次元回転 / ホイール=ズーム / 1–4=多胞体 / F=ファイバー / P=多胞体 / D=塵 / Space=静止
