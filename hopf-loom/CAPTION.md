# HOPF LOOM — S³ の織物

CCF5 Series / 04
2026 年 / リアルタイム生成アート(WebGL2・単一 HTML)

私たちの目は三次元までしか見えない。それでも数学は、四次元の球面 S³ が無数の円で隙間なく編み上げられていること——Hopf ファイブレーション——を知っている。本作は、その見えないはずの織物をステレオ投影によって三次元へ降ろし、入れ子のトーラス群が絶えず変形しながら呼吸する様を織機(loom)のように描く。

そこに 120 の頂点と 720 の辺をもつ四次元正多胞体・正六百胞体が重なり、四次元的な二重回転を続けながら、辺が膨らんだ曲線となって画面を貫く。極を通過する瞬間、ファイバーは無限遠へと閃光を放つ。多次元世界は遠い抽象ではなく、いま目の前で編まれ続けている。

- ドラッグ:視点 / Shift+ドラッグ:4 次元回転 / ホイール:ズーム
- 1–4:正五胞体 / 正八胞体 / 正二十四胞体 / 正六百胞体
- F/P/D:レイヤー表示 / Space:静止
